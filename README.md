# 集成方式 

由于项目还在初步阶段 并未上传pub 请直接下载集成 

1. 将项目clone到本地

2. 在pubspec.yaml中添加

   ```dart
     lfklinewidget:
       path: 你下载的项目文件绝对路径 如C:\gitProgram\lf_kline_widget
   ```

   

​    3. 项目中有使用示例  你也可以在我的flutter网站项目中看我是如何使用的 [戳这里](https://github.com/SOPD/lf_flutter_website)
